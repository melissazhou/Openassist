# 08 前端 UI 规范

## 设计系统: shadcn/ui + Tailwind CSS

### 主题
- 亮色为主，支持暗色切换
- 主色: Slate (专业) + Blue (操作/链接)
- 成功: Green | 警告: Amber | 错误: Red

## 核心页面设计

### 1. Dashboard
- **KPI卡片:** 待处理数 | 本周新增 | 平均处理时间 | SLA达标率
- **状态分布饼图** + **分类柱状图**
- **近期活动流** (Timeline)
- **待办事项** (我的待处理)

### 2. 变更请求列表
- **TanStack Table** 高级表格
  - 列排序/过滤/隐藏/调整宽度
  - 行选择 (批量操作)
  - 内联状态标签 (彩色Badge)
- **高级筛选栏** (可保存筛选条件)
- **快速搜索** (Command+K 全局搜索)
- **视图切换:** 表格 / 看板(Kanban) / 时间线

### 3. 变更请求详情
- **Header:** 编号+标题+状态Badge+优先级+操作按钮
- **Tab页:**
  - 详情 (字段信息+变更内容对比)
  - 审批流程 (可视化进度条)
  - 评论 (对话式+内部备注)
  - 附件
  - 同步记录
  - 变更历史 (字段级Diff)

### 4. 审批页面
- 待审批列表 + 一键批量审批
- 详情内嵌审批操作 (通过/驳回/退回+评论)

### 5. 数据质量仪表板
- 质量评分仪表盘 (Gauge)
- 错误热力图 (按字段×规则)
- 问题数据列表 (可直接跳转修正)

## 交互规范

| 场景 | 组件 |
|---|---|
| 确认操作 | AlertDialog (shadcn) |
| 表单 | Sheet/Drawer (侧滑) 或 独立页面 |
| 通知 | Toast (右上角) |
| 加载 | Skeleton (骨架屏) |
| 空状态 | 插图+提示文字+操作按钮 |
| 全局搜索 | Command Palette (Cmd+K) |
| 数据导出 | DropdownMenu (Excel/CSV/PDF) |

## 响应式

- Desktop优先 (1280px+)
- Tablet适配 (768px-1279px): 侧边栏折叠
- 移动端: 基本可用但非重点

## 国际化

- 默认中文，支持英文切换
- 使用 `next-intl`
- 日期格式: YYYY-MM-DD (统一)
